Dapp Lottery

Dapp lottery is a dapp that explores the use of smart contracts and chainlink tools to create a trustless lottery. This dapp was built on rinkeby testnet, we are using fake ethereum.

Before you interact with this dapp you have to connect your metamask. If you do not have a metamask installed go to this tutorial to learn how to use it.
